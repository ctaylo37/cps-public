Kafka

Apache open source distributed publish-subscribe messaging system for handling real-time data feeds. Kafka is a high throughput, fault-tolerant, partitioned and replicated commit log service.


Launch

99.922% MTD

Mission

Build highly available, fully managed and cost efficient messaging platform adaptable to PayPal needs and running on a hybrid cloud.

SLA

  • Availability: 99.995%
  • Latency: < 10ms at 95th percentile
  • Data loss: < 0.02%

Core Features

  • High throughput and horizontally scalable (through partition sharding and distributed consumption).
  • Supports both single-queue and publisher/subscriber (multiple listeners) models.
  • Intra cluster replication for fault-tolerance.
  • Persistent to disk for durability.
  • Supports replication of data across data centers.
  • Supports both real-time and batch consumption.
